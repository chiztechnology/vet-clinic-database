
# Vet clinic postgress-database
This is a frist Postgress project that contains a basic database structure of a Vect clinic venture

## Getting Started

This repository includes files with plain SQL that can be used to recreate a database:

-  [schema.sql](./schema.sql) contains the  `create` tables instructions.
-  [data.sql](./data.sql) contains instructions and sample data to populate tables.
-  [queries.sql](./queries.sql) contains instructions and queries to manipulate the data. 

**Important note: this file might include queries that make changes in the database (e.g., remove records). Use them responsibly!**
